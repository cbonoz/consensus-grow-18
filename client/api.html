Add an Item:

  
  ===
      Run a Stimming prediction based on provided data
  ===

  Definition:
  
  POST /api/items/add
  

  Example Request Body: 


  
 {"items" :[ { "name": "Swiss Chard/Green", // name of the item (not necessarily unique) "unit": "12 BU", // quantity or
size of the item "metadata": "Product of USA", // additional comments or info on this item "uuid": XXXX, // uuid of the item
"origin": Satur Farms NY 11935" "packDate": "05-08-2018" }, ... ] }



                            
  Example Response: 
  // Successful DB response.
  
  {
      inserted: 1,
  }

Record delivery of an item:

  
  ===
  Record the delivery of an item at a location
  ===

  Definition:
  
  POST /api/item/delivery
  

  Example Response:
  
  {
      userId: XXXXXXX // database-assigned userId.
  }
  

  Example Response:
  // Successful DB response.
  
    Recored a delivery for a particular item id - validated by ChainPoint.
  {
      "itemId": XXXX, // uuid of the received item.
      "locationId": XXXXX, // uuid of the receiving location
      "latitude": XXXX, // latitude of the receiver
      "longitude": XXX // longitude of the receiver
      "timestamp": XXXXX // time of delivery
  }

Get Item History:

  
  ===
    Get Validated Supply chain history for item.
  ===

  Definition:

  
  GET /api/item/history/:itemId
  

  Example Response:
  // Successful DB response.

[
  ...list of deliveries for the particular item id.
]

Verify a Delivery Record:

  
  ===
    Compare delivery records against history using Tierion and Chainpoint (used internally to the API).
  ===
  

  Definition:
  
  POST /api/record/verify
  

  Example Request Body:
  
  {
    ...record body
  }
  

  Example Response:
  
  {
      status: 1, // 1 if valid, 0 if no match.
  }
